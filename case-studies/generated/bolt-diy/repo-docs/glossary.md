# Glossary

| Term | Plain meaning | Further reading |
| --- | --- | --- |
| Context buffer | The small selected file set inserted into the build prompt instead of the whole project. | [context buffer](modules/context-buffer.md) |
| `SwitchableStream` | The route-level stream wrapper used when a long response needs continuation segments. | [chat to stream](walkthroughs/chat-to-stream.md) |
| MCP tool | A tool exposed by an MCP server and surfaced to the model without direct execution until approved. | [MCP tool loop](modules/mcp-tool-loop.md) |
| `toolsWithoutExecute` | Tool declarations copied without execute functions so model calls can describe tools safely. | [MCP tool loop](modules/mcp-tool-loop.md) |

Evidence status: Confirmed unless noted.
