# Glossary

| Term | Plain meaning | Further reading |
| --- | --- | --- |
| `Coder` | The session object that owns model calls, active files, edits, repo map, linting, tests, and commits. | [CLI session model](modules/cli-session-model.md) |
| `GitRepo` | Aider's wrapper around GitPython for repository discovery, tracked files, diffs, and commits. | [Git safety](modules/git-safety.md) |
| Repo map | A compact source map sent to the model when the coder and model support it. | [terminal edit run](walkthroughs/terminal-edit-run.md) |
| Dirty commit | A commit made before edits so local changes can be protected before the model writes. | [Git safety](modules/git-safety.md) |

Evidence status: Confirmed unless noted.
